# Acadly Super Admin Dashboard

## Project Overview
Acadly is an educational platform for:
- students
- parents
- supervisors
- admins
- super admins

This dashboard is for SUPER ADMIN only.

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Firebase
- Firestore
- Firebase Authentication
- Recharts

## Dashboard Features
- Authentication
- Protected Routes
- Users Management
- Centers Management
- Analytics
- Notifications
- Payment Monitoring
- Real-time Firestore updates
